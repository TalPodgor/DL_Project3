================================================================================
PROJECT OVERVIEW
================================================================================

Project: BGU Deep Learning Final Project - Project 3
Title: Synthetic-to-Real Image Translation for Chessboard Rendering
Goal: Train a CUT (Contrastive Unpaired Translation) model to transform
      synthetic Blender-rendered chess images into realistic-looking images.

The user has:
- Real chess images extracted from video recordings of 5 games (games 2, 4, 5, 6, 7)
- Synthetic chess images generated using Blender from FEN positions

================================================================================
DATASET STRUCTURE
================================================================================

Dataset organization (CUT format):
    dataset/
    ├── trainA/    (2,664 synthetic images - games 4,5,6,7, white+black viewpoints)
    ├── trainB/    (736 real images - games 4,5,6,7, white+black viewpoints)
    ├── testA/     (462 synthetic images - game 2)

Train/Test split strategy:

- Testing: Game 2

================================================================================
SCRIPTS CREATED
================================================================================

1. preprocess_real_images.py
   - Collects real images from gameX_per_frame/tagged_images/
   - Creates 180° rotated versions (black viewpoint)
   - Output: dataset/real/

2. preprocess_synthetic_images.py
   - Collects synthetic images from synthetic_gameX/cropped/
   - Deletes comparison.png files
   - Creates 180° rotated versions
   - Output: dataset/synthetic/

3. organize_cut_dataset.py
   - Organizes images into trainA/trainB/testA/testB structure
   - Game-based split (train: 4,5,6,7 | test: 2)

4. cluster_setup.sh
   - One-time setup script for BGU cluster
   - Clones CUT repository
   - Installs PyTorch and dependencies

5. train_cut.sh
   - Training script with parameters:
     --batch_size 4
     --crop_size 256
     --n_epochs 200
     --n_epochs_decay 200
     --save_epoch_freq 20

6. resume_training.sh
   - Detects last saved epoch from checkpoint files
   - Resumes training with correct --epoch_count
   - Use this after session disconnects

7. test_cut.sh
   - Runs inference on test set after training completes

Model & architecture:
- Framework:    CUT (Contrastive Unpaired Translation), CUT_mode = CUT

- Discriminator: PatchGAN ('basic'), 3 layers, ndf=64, instance norm
- NCE head F:   mlp_sample, 256-D, 256 patches per layer
- NCE layers:   0, 4, 8, 12, 16 (5 feature layers from G's encoder)

- lambda_GAN: 1.0
- lambda_NCE: 1.0

Optimizer & schedule:
- Adam (beta1=0.5, beta2=0.999), lr=2e-4
